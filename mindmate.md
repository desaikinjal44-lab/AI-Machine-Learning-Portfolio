# MindMate – Stress Recognition Quiz Chatbot

## Introduction

MindMate is an AI-powered chatbot that I created during the AI Lab. It helps users recognize common signs of stress through an interactive quiz and provides educational suggestions for managing stress.

## Objective

To create an AI chatbot that helps users become more aware of their stress levels and encourages healthy coping strategies.

## Audience

This project is intended for instructors, employers, and anyone interested in how AI can be used to solve real-world problems.

## Process

I used the Design Thinking process:

- Empathize
- Define
- Ideate
- Prototype
- Test

I built the chatbot using Mizou and tested it with different user responses.

## Tools Used

- Mizou
- ChatGPT
- STORM AI
- Generative AI

## Value Proposition

This project demonstrates my ability to apply AI and design thinking to solve a real-world problem while creating a user-friendly chatbot.
